# Run

Run server
>npm run build

Run server (dev)
>npm run serve


# Setup

### Install client dependencies
>npm install

  * Contact Email
  * Assiatance Quater
  * Duration of Course
  * College Name
  * Reference 1 Name
  * Reference 2 Name
  * Notes
* See specific application
  * All details
  * Application Status
  * Referral Status
  * Notes
* Repayment applications for year/quater or before
